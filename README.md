# Live chat on rails.

- This is a test project for a Live chat.

## Features

- Account settings.
- Simple secure and anonymous registration system.
- Different chatrooms.
- Live updates with ajax and `ActionCable` (websockets).
- Fully responsive and mobile ready.

## Planned features


- Admin moderation.
